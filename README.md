# ООО «СтройМатериалы» — демоэкзамен

WPF-приложение (C# / .NET 8) с базой данных PostgreSQL для магазина строительных материалов.

## Требования

- [.NET 8 SDK](https://dotnet.microsoft.com/download)
- PostgreSQL 14+

## Установка БД

1. Создайте базу:
   ```sql
   CREATE DATABASE stroymaterialy ENCODING 'UTF8';
   ```
2. Выполните скрипт `database/init.sql` в этой базе.
3. Укажите строку подключения в `StroyMaterialy/appsettings.json`:
   ```json
   "Postgres": "Host=localhost;Port=5432;Database=stroymaterialy;Username=postgres;Password=ВАШ_ПАРОЛЬ"
   ```

## Запуск

```bash
cd StroyMaterialy
dotnet run
```

При первом запуске приложение предложит импортировать данные из папки `импорт` (товары, пользователи, заказы, пункты выдачи).

| Роль | Товары | Заказы |
|------|--------|--------|
