# ООО «СтройМатериалы» — демоэкзамен

WPF-приложение (C# / .NET 8) с базой данных PostgreSQL для магазина строительных материалов.

## Требования

- [.NET 8 SDK](https://dotnet.microsoft.com/download)
- PostgreSQL 14+

## Установка БД

1. Создайте базу:
   ```sql
   CREATE DATABASE stroymaterialy ENCODING 'UTF8';
   ```
2. Выполните скрипт `database/init.sql` в этой базе.
3. Укажите строку подключения в `StroyMaterialy/appsettings.json`:
   ```json
   "Postgres": "Host=localhost;Port=5432;Database=stroymaterialy;Username=postgres;Password=ВАШ_ПАРОЛЬ"
   ```

## Запуск

```bash
cd StroyMaterialy
dotnet run
```

При первом запуске приложение предложит импортировать данные из папки `импорт` (товары, пользователи, заказы, пункты выдачи).

## Импорт / экспорт Excel (все таблицы)

Доступно **администратору** на главной форме: кнопки **«Экспорт Excel»** и **«Импорт Excel»**.

Один файл `.xlsx` содержит листы:

| Лист | Таблица БД |
|------|------------|
| `roles` | роли |
| `users` | пользователи |
| `pickup_points` | пункты выдачи |
| `products` | товары |
| `orders` | заказы |
| `order_items` | позиции заказов (артикул + номер заказа) |

**Экспорт** — сохраняет текущие данные из PostgreSQL в выбранный файл.

**Импорт** — загружает все листы из файла (заменяет данные в таблицах; роли только дополняются).  
Картинки товаров ищутся в той же папке, что и Excel-файл (имена из колонки `image_file`).

При первом запуске по-прежнему можно импортировать демо-данные из папки `импорт` (отдельные файлы).
